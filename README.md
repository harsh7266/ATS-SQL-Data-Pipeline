# ATS Data Pipeline & Advanced SQL Analytics 🛠️

## 📌 Project Overview
Real-world data is rarely clean. This project demonstrates advanced data engineering and SQL analytics by processing 1,500 highly unstructured, messy job postings scraped from competitor websites. The objective was to load unstructured text into a relational database and use Advanced SQL to clean HTML tags, standardize salary formats, and extract trending tech skills.

## 🛠️ Tech Stack & Tools
* **Database Management:** SQLite
* **Scripting & Automation:** Python (Pandas)
* **Advanced SQL Concepts Applied:** * Common Table Expressions (CTEs)
  * Window Functions (`RANK() OVER`)
  * String Manipulation (`TRIM`, `UPPER`, `SUBSTR`, `REPLACE`)
  * Control Flow (`CASE WHEN`)

## 📂 Files in this Repository
* `project2_sql.ipynb`: The Jupyter Notebook containing the Python-to-SQL ingestion script and the Advanced SQL queries.
* `QuickHyre_Messy_Job_Postings.csv`: The raw, unstructured dataset containing missing values, varied text cases, and HTML tags.
* `quickhyre_jobs.db`: The local SQLite database generated by the pipeline.
* `screenshots`

## 🔍 SQL Query Highlights
### 1. Data Standardization & Cleaning
Used a CTE combined with `CASE` statements to parse wildly varying salary string formats (e.g., "$80k - $100k" vs. "80000-100000") into a clean, standardized `min_salary_usd` integer column. Used nested `REPLACE` functions to strip raw HTML tags (`<p>`, `<ul>`, `<li>`) from job descriptions.

### 2. Window Functions for Salary Ranking
Implemented `RANK() OVER(PARTITION BY company_name ORDER BY min_salary_usd DESC)` to instantly identify the highest-paying roles within each specific company grouping.

### 3. Keyword Extraction
Utilized `LIKE` operators and aggregate `SUM` functions to scan the cleaned text blocks, quantifying the market demand for specific skills (Python, SQL, Power BI, Tableau) across all 1,500 postings.

## 💡 Key Findings
* Successfully transformed 100% of salvageable raw text into a queryable, structured SQL schema.
* Extracted and ranked competitor salary bands to provide competitive intelligence for HR compensation strategies.
